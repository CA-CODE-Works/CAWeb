CAWeb utilizes GULP to compile all CSS & JS files for the theme, there are several gulp tasks available in the theme. Make sure npm is installed along with necessary packages, from a terminal in the projects root directory run the commands below. 

npm -v
npm install
gulp -v

gulp commands

Note: The following commands allow for 2 parameters, if no parameter is supplied then both minified/unminified files are built.

--dev
- Builds unminified files. 
--prod
- Builds minified files. 
gulp admin-css
- Builds admin CSS files. This CSS file is used on the CAWeb Theme administrative menus only. 
gulp frontend-css
- Builds individual cagov-version-colorscheme files. 
gulp bootstrap-css
- Builds admin bootstrap CSS file. This CSS file is used to style the CAWeb Theme administrative menus only. 
gulp admin-js
- Builds admin JS file. This JS file is used on the CAWeb Theme administrative menus only. 
gulp caweb-js
- Builds individual caweb-version JS files. Main JS file loaded when State CDN option is not enabled. 
gulp bootstrap-js
- Builds admin bootstrap JS file. This JS file is used to enable bootstrap scripts in the CAWeb Theme administrative menus only. 
gulp customizer-js
- Builds theme customizer and theme customizer control JS file. These JS files are loaded when using the WordPress Appearance->Customize feature. 
gulp build
- Builds all of the CSS & JS theme asset files.

Note: The following commands do not accept parameters.

gulp dev
- Builds all of the CSS & JS theme asset files unminified. 
gulp prod
- Builds all of the CSS & JS theme asset files minified.

Note: The following commands allow for 1 parameter. If parameter is not passed then all PHP files will be beautified.

--file
- HTML/PHP file to beautify. 
gulp beautify
- Beautify specific file if parameter is passed, otherwise beautify all PHP files.
View all gulp tasks 
gulp --tasks
