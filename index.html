The CAWeb Theme is an open source WordPress Theme offered by the California Department of Technology to state agencies and departments within the State of California and fully integrates the California State Template.

CAWeb leverages GitHub to provide for better collaboration and and encourages participation.

Tools
  • NPM
    • GULP - used to compile all CSS & JS files for the theme.
  • Composer
    • WPCS - used to enforce WordPress Coding Standards.
    • phpDocumentor - used to document the theme.

WordPress Code Enforcement

CAWeb utilizes WPCS to enforce WordPress Coding Standards. Make sure composer is installed along with necessary packages, from a terminal in the projects root directory run the commands below. For information on how to enforce the standards, see here. 

composer --version
composer install --no-dev
.\vendor\bin\phpcs --version
.\vendor\bin\php-cs-fixer --version
.\vendor\bin\phpcbf --version
Compiling CSS/JS

CAWeb utilizes GULP to compile all CSS & JS files for the theme. Make sure npm is installed along with necessary packages, from a terminal in the projects root directory run the commands below. For more CAWeb Theme gulp tasks, see here. 

npm -v
npm install
gulp -v
